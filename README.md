# Cost Change UI (Excel mockup -> Next.js)

This is a small prototype that converts your Excel mockup into a working web UI using:

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Client-side mock data (no backend yet)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this project to GitHub
2. In Vercel: New Project -> Import GitHub repo -> Deploy

No special settings needed.
